数组拷贝

System.arraycopy()

它可以复制基本类型数组和对象数组。 复制对象数组时，只是复制了对象的引用，而不是对象本身的拷贝。

它不会执行自动装箱和自动拆箱，两个数组必须具有相同的确切类型。
