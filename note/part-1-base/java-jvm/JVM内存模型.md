# JVM的内存模型

JVM内存分为五大区域：
* 堆
* 方法区
* 虚拟机栈
* 本地方法栈
* 程序计数器

其中，**堆**和**方法区**是所有线程共享的数据区，**虚拟机栈**、**本地方法栈**和**程序计数器**是线程隔离的数据区。
